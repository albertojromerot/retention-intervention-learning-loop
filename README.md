# Synthetic Voluntary Attrition Intervention Learning Loop

A clean-room Python project demonstrating a synthetic, governed machine learning workflow for voluntary attrition prediction, intervention prioritisation, treatment logging, and batch re-learning in financial services.

## Project status

Initial scaffold created. No synthetic data has been generated yet.

## Core idea

The project will compare three decision approaches:

1. A rules-based baseline.
2. A first machine learning model.
3. A treatment-aware machine learning model retrained after synthetic intervention outcomes.

## Planned workflow

The project will generate synthetic financial-services customer behaviour, estimate voluntary attrition risk over a future 90-day window, prioritise eligible customers for intervention, simulate treatment outcomes, and retrain a later model version using the synthetic treatment log.

## Data statement

This project uses fully synthetic data only.

No real customer, member, employer, financial, operational, or proprietary data is included.
