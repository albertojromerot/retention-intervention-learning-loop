# Data Provenance

All datasets used by this project are generated synthetically by the code in this repository.

The synthetic data is designed to represent plausible financial-services behaviour for demonstration purposes only.

No external, confidential, employer, customer, member, operational, financial, or proprietary datasets are used.

## Planned synthetic data layers

The project will generate synthetic data for:

- Customer profiles.
- Monthly relationship activity.
- Digital engagement.
- Product relationship depth.
- Credit relationship signals.
- Service interaction signals.
- Process friction signals.
- Voluntary attrition labels.
- Intervention assignments.
- Treatment logs.
- Post-intervention outcomes.

## Reproducibility

Synthetic data generation will use deterministic random seeds so that results can be reproduced locally.
